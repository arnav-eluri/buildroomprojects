BuildRoom Projects are simple, hands-on learning projects designed to support each main learning module. They help learners apply concepts through practical implementation instead of relying only on theory. Each project is aligned with a specific topic and focuses on building real-world applications in a step-by-step manner. 

The ultimate goal? To improve your core understanding, sharpen your problem-solving abilities, and build the confidence you need to create amazing software systems from scratch.

## Why BuildRoom Projects?

- **Learn by Doing**: We believe the best way to learn code is to write code.
- **Step-by-Step Progression**: Every project starts simple and gradually introduces more advanced concepts.
- **Real-World Ready**: Instead of isolated exercises, you build actual applications that you can use and expand upon.

## Repository Structure

Inside this repository, you'll find different project folders. Each folder represents a standalone project or module and comes with its own instructions.

### Projects

| Folder | Description |
|--------|-------------|
| [`smartchatbot`](./smartchatbot/) | Two Gemini-powered chatbots — a general-purpose assistant and a college FAQ bot. Requires a Google Gemini API key. |
| [`debugbot`](./debugbot/) | A code debug assistant that analyzes your Python code for bugs and a practice mode with random buggy code challenges. Fully offline — no API key needed. |

## How to Get Started

1. **Browse the Projects**: Look through the folders in this repository to find a project that interests you.
2. **Read the Project README**: Every project folder contains its own `README.md` file with specific instructions, prerequisites, and setup guides.
3. **Write Code**: Follow the steps, tinker with the code, and don't be afraid to break things! That is how we learn.

Happy coding!
